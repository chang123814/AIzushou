# AI助手 微信小程序

这是一个多功能AI助手微信小程序项目，当前的核心功能是实现了一个完整的社区问答模块。

## ✨ 主要功能

- **底部Tab导航栏**:
  - 包含「首页」、「功能」、「社区」、「我的」四个主要入口，结构清晰。
  - Tab栏图标和文本均已配置完成。

- **社区问答模块 (纯前端实现)**:
  - **问题列表页 (`pages/community/index`)**:
    - 使用卡片形式展示问题列表。
    - 支持按「热门」和「最新」进行切换排序。
    - 顶部包含一个搜索框 (样式已完成)。
    - 右下角有悬浮按钮，用于发布新问题。
  - **问题发布页 (`pages/community/post/index`)**:
    - 提供一个完整的表单，用户可以输入问题标题、内容和分类标签。
    - 包含提交校验和模拟成功反馈。
  - **问题详情页 (`pages/community/detail/index`)**:
    - 展示问题的完整标题、内容、作者信息和标签。
    - 实现了**多级回复**的列表展示，包括嵌套回复的样式。
    - 用户可以进行模拟的点赞、收藏和回复操作。
    - 底部有固定的回复输入框，提供流畅的交互体验。

- **模块化与可扩展性**:
  - 所有页面的功能和API请求都已在对应的 `.js` 文件中解耦。
  - 预留了清晰的 `wx.request` 接口位置，方便后续快速接入真实后端服务器。
  - 使用本地模拟数据 (Mock Data) 完整地支撑了整个前端功能的交互流程。

## 📂 项目结构

```
.
├── tubiao/               # 存放Tab栏图标
├── pages/                # 小程序所有页面
│   ├── home/             # 首页
│   ├── index/            # 功能页
│   ├── mine/             # 我的页面
│   └── community/        # 社区模块
│       ├── index.js      # -> 社区首页逻辑
│       ├── index.wxml
│       ├── index.wxss
│       ├── post/         # -> 发布问题页
│       └── detail/       # -> 问题详情页
├── app.js
├── app.json              # -> 全局配置，包含页面和Tab栏注册
├── app.wxss
└── project.config.json
```

## 🚀 如何运行

1.  将项目代码克隆或下载到本地。
2.  打开**微信开发者工具**。
3.  选择「导入」，并指向本项目所在的根目录。
4.  AppID 可使用测试号。
5.  点击「编译」即可在模拟器或真机上预览效果。

## 🔮 后续计划

当前项目已完成问答模块的**纯前端**部分。下一步的核心工作是将前端与后端服务打通。

- **后端对接**:
  - 在各个页面的 `.js` 文件中，找到被注释掉的 `wx.request` 代码块。
  - 将 `YOUR_API_ENDPOINT` 替换为你的实际服务器接口地址。
  - 根据你的API规范，调整请求参数和返回数据的处理逻辑，即可替换掉现有的模拟数据。
